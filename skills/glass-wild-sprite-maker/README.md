# Glass Wild Sprite Maker

ChatGPT에서 Glass Wild 도트 스프라이트를 만들 때 쓰는 스킬입니다. 규칙은 전부 [`SKILL.md`](SKILL.md)에 있습니다.

## 설치

1. `glass-wild-sprite-maker` 폴더를 ZIP으로 압축 (루트에 `SKILL.md`)
2. ChatGPT Skills → Upload

## 예시

```text
네리트 달팽이 이동 애니메이션 만들어줘
```

```text
육상 코르크 벽지 타일 만들어줘
```

```text
등각류 Idle Walk Eat Avoid 만들어줘
```
