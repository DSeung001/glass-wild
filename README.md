# Glass Wild

Godot 기반의 도트 스타일 관찰형 사육 시뮬레이션 게임입니다.

플레이어는 여러 개의 사육장을 프리셋 기반으로 구성하고, 생물별 환경 조건과 합사 관계를 관리하여 안정적인 미니 생태계를 만듭니다. 일반 플레이 화면뿐 아니라 컴퓨터 배경화면 위에서 생태계를 관찰하는 배경화면 모드를 지원하는 것을 목표로 합니다.

## 핵심 질문

> 어떤 생물을 어느 사육장에 배치하고, 어떤 환경을 만들어야 모두가 안전하게 살아갈 수 있는가?

> 플레이하지 않을 때에도 이 생태계를 계속 보고 싶게 만드는가?

## MVP 원칙

- 모든 사육장, 생물, 식물, 장비는 처음부터 사용 가능합니다.
- 해금, 재화, 상점, 뽑기보다 환경 설계와 합사 판단을 우선합니다.
- 사육장 3개를 독립적으로 운영합니다.
- 포식, 경쟁, 과밀, 환경 부적합을 명확하게 경고합니다.
- 사고는 즉시 게임 오버가 아니라 복구 플레이로 연결합니다.
- UI 없이도 도트 애니메이션으로 생물 상태를 어느 정도 파악할 수 있어야 합니다.
