# MVP 식물 역할과 데이터

## 목적

식물은 장식만이 아니라 습도, 은신처, 시야 차단과 활동 공간에 영향을 주는 환경 구성 요소입니다.

## 공통 데이터 필드

| 필드 | 설명 |
|---|---|
| ID | 고유 식별자 |
| 표시 이름 | UI 이름 |
| 역할 태그 | 습도 유지, 은신처, 벽면, 먹이 등 |
| 권장 온도 | 식물 상태 판정 |
| 권장 습도 | 식물 상태 판정 |
| 조명 요구 | 부족, 적정, 강함 |
| 바닥재 요구 | 허용 가능한 바닥재 태그 |
| 공간 점유 | 배치와 밀도 판정 |
| 환경 효과 | 습도·청결·은신처 등에 미치는 영향 |
| 성장 크기 | 시야와 활동 구역 변화 |
| 섭식 대상 여부 | 초식 생물이 먹을 수 있는지 |

## MVP 역할군 6종

| ID | 역할명 | 주요 효과 | 주요 부담 |
|---|---|---|---|
| moisture_plant | 습도 유지 식물 | 습도 감소 속도 완화 | 과습 시 약화 가능 |
| ground_cover | 바닥 덮개 식물 | 작은 생물 은신처 제공 | 바닥 활동 공간 감소 |
| tall_shelter | 높은 은신처 식물 | 시야 차단과 상부 은신 | 공간 점유 큼 |
| climbing_plant | 벽면 등반 식물 | 벽면 활동 구역 확대 | 높은 식물 밀도 유발 가능 |
| dry_plant | 건조 환경 식물 | 건조 사육장 안정화 | 고습에서 약화 |
| edible_plant | 먹이 제공 식물 | 초식 생물 먹이 보조 | 반복 섭식 시 효과 감소 |

## 식물 상태

| 상태 | 조건 | 게임 효과 |
|---|---|---|
| 건강 | 환경 적합 | 환경 효과 100% |
| 약화 | 일시적 부적합 | 환경 효과 50~75% |
| 시듦 | 장시간 부적합 | 환경 효과 최소, 경고 발생 |

MVP에서는 식물 사망, 번식과 복잡한 성장 단계를 제외합니다.

## 설계 원칙

- 식물의 장점에는 반드시 공간 또는 환경상의 부담이 있어야 합니다.
- 한 식물이 모든 문제를 해결하지 않도록 역할을 분리합니다.
- 생물의 행동 구역과 연결되는 효과를 우선합니다.
- 식물 상태는 색 변화와 잎의 움직임 감소로 표현합니다.
- 도트 애니메이션은 미세한 흔들림 위주로 최소 제작합니다.

## MVP 애니메이션

- 기본 흔들림
- 약화 상태
- 섭식 반응이 필요한 식물만 짧은 반응 프레임
