# MVP 완료 체크리스트

## 문서와 결정

- [ ] 핵심 콘셉트와 비목표가 최신 상태이다.
- [ ] `docs/decisions/`의 P0 결정이 모두 확정되었다.
- [ ] 실제 생물 정보를 사용할 경우 검수 출처와 책임자가 정해졌다.
- [ ] Figma 핵심 흐름 5개가 연결되어 있다.
- [ ] 도트 스타일 가이드와 기준 셀 크기가 확정되었다.

## 사육장

- [ ] 사육장 3개를 독립적으로 운영할 수 있다.
- [ ] 사육장 사이를 빠르게 전환할 수 있다.
- [ ] 각 사육장의 환경, 생물과 배치물이 독립 저장된다.
- [ ] 프리셋 변경 전 예상 위험을 확인할 수 있다.
- [ ] 생물을 다른 사육장으로 이동할 수 있다.

## 환경

- [ ] 온도, 습도, 청결도와 먹이 상태가 시간에 따라 변한다.
- [ ] 생물별 환경 적합도가 표시된다.
- [ ] 수용량과 은신처 부족을 판정한다.
- [ ] 장비와 아이템의 장점·부작용이 모두 적용된다.
- [ ] 환경 변화가 생물 행동에 반영된다.

## 생물

- [ ] MVP 대상 생물이 정의된 역할 차이를 보인다.
- [ ] 이동, 섭식, 은신 또는 휴식 행동이 동작한다.
- [ ] 건강, 배고픔과 스트레스 상태가 존재한다.
- [ ] 행동 전환이 시각적으로 끊기지 않는다.
- [ ] 상태가 저장되고 불러오기 후 복구된다.

## 관계

- [ ] 안전, 주의, 위험 합사 판정이 동작한다.
- [ ] 먹이 경쟁과 과밀을 감지한다.
- [ ] 명확한 포식 관계가 경고를 발생시킨다.
- [ ] 공격 전에 관심, 접근과 추적 단계가 보인다.
- [ ] 플레이어가 사고 전에 분리하거나 격리할 수 있다.
- [ ] 사고는 복구 가능한 부상 또는 탈진으로 연결된다.

## UI/UX

- [ ] 대시보드에서 가장 위험한 사육장을 즉시 찾을 수 있다.
- [ ] 배치 전에 환경과 합사 위험을 확인할 수 있다.
- [ ] 모든 경고가 대상, 원인과 해결책을 포함한다.
- [ ] 관찰 화면의 UI가 생물 행동을 과도하게 가리지 않는다.
- [ ] 긴급 상황에서 즉시 행동이 우선 표시된다.
- [ ] 색상 없이도 상태를 구분할 수 있다.

## 도트 에셋

- [ ] 모든 생물이 동일한 시점과 광원 규칙을 따른다.
- [ ] 프레임별 바닥 접점과 피벗이 고정되어 있다.
- [ ] 정수 배율에서 픽셀이 흐려지지 않는다.
- [ ] 좌우 반전 또는 방향별 스프라이트 규칙이 일관된다.
- [ ] AI 생성 원본과 수동 수정본의 기록이 남아 있다.
- [ ] 사용 도구와 참고 자료의 라이선스가 기록되어 있다.

## 배경화면 모드

- [ ] 일반 플레이와 동일한 저장 데이터를 사용한다.
- [ ] UI를 최소화한 관찰 상태로 진입할 수 있다.
- [ ] 일반 입력과 인터랙션 입력을 전환할 수 있다.
- [ ] 주의와 긴급 상태가 방해가 적은 방식으로 표시된다.
- [ ] 일반 화면으로 안정적으로 복귀할 수 있다.
- [ ] 다중 모니터와 화면 배율을 테스트했다.
- [ ] 장시간 실행 성능을 측정했다.

## 저장과 복구

- [ ] 사육장 환경이 저장된다.
- [ ] 배치된 생물, 식물, 장비와 구조물이 저장된다.
- [ ] 생물 상태와 경고가 저장된다.
- [ ] 비정상 종료 후 저장 손상에 대한 최소 대응이 있다.
- [ ] 버전이 다른 저장 데이터에 대한 마이그레이션 전략이 있다.

## 플레이테스트

- [ ] 사용자가 설명 없이 위험 사육장을 찾는다.
- [ ] 사용자가 위험한 생물 조합을 알아차린다.
- [ ] 사용자가 습도 문제를 해결한다.
- [ ] 사용자가 사고 전 생물을 분리한다.
- [ ] 사용자가 사육장 간 이동 결과를 이해한다.
- [ ] 사용자가 배경화면 모드를 계속 켜두고 싶다고 평가한다.

## 출시 차단 조건

다음 항목 중 하나라도 충족되지 않으면 MVP 배포를 중단합니다.

- 저장 데이터가 반복적으로 손상됨
- 경고 원인과 실제 판정이 일치하지 않음
- 포식 사고가 경고 없이 발생함
- 입력 통과 실패로 다른 앱 사용을 방해함
- 도트 에셋의 프레임 흔들림이 핵심 행동을 알아보기 어렵게 함
- 핵심 플레이 흐름이 진행 불가능한 오류가 있음
