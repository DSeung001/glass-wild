# MVP 범위

## 목표

MVP는 다음 두 가설을 검증합니다.

1. 여러 사육장의 환경과 생물 조합을 안정화하는 과정이 재미있는가?
2. 플레이하지 않을 때에도 도트 생태계를 관찰하고 싶은가?

## 포함 범위

### 사육장

- 독립적인 사육장 3개
- 고습, 온대, 건조 프리셋
- 프리셋 변경
- 사육장 간 생물 이동
- 사육장별 독립 환경 상태

### 환경

- 온도
- 습도
- 청결도
- 먹이 상태
- 수용량
- 은신처 상태
- 바닥재 적합성

### 생물

- 역할이 다른 생물 6종
- 이동, 섭식, 은신 또는 휴식
- 건강, 배고픔, 스트레스
- 환경 적합도
- 안전, 주의, 위험 합사 판정
- 포식 징후, 추적, 공격 예고
- 부상과 임시 격리

### 식물과 아이템

- 역할형 식물 6종
- 바닥재 4종
- 구조물 4종
- 환경 장비 5종
- 먹이 범주 4종
- 청소, 이동, 격리 관리 행동

### UI

- 시작 화면
- 전체 사육장 대시보드
- 개별 사육장 관찰
- 가꾸기 모드
- 생물 목록
- 배치 전 진단
- 생물 상세 패널
- 긴급 오버레이
- 설정

### 모드

- 자유 모드
- 학습용 시나리오 4개
- 배경화면 모드

### 시간과 저장

- 일시정지, 1배, 2배, 4배
- 긴급 자동 감속
- 사육장과 생물 상태 저장
- 배경화면 설정 저장

## 모든 콘텐츠 공개 원칙

MVP에서는 다음을 사용하지 않습니다.

- 콘텐츠 해금
- 플레이어 레벨
- 상점
- 재화
- 희귀도
- 확률형 보상

콘텐츠 선택의 부담은 필터, 추천 사육장과 배치 진단으로 줄입니다.

## 제외 범위

- 번식, 알, 유충과 성장 단계
- 유전 특성
- 복잡한 질병과 약품 치료
- 사실적인 사망 표현
- 계절과 날씨
- 현실 시간 연동
- 오프라인 시간 진행
- 온라인 기능
- 거래와 멀티플레이
- 업적 보상
- 모바일과 콘솔 배포
- 고급 지역별 환경 시뮬레이션

## MVP 콘텐츠 목표량

| 항목 | 수량 |
|---|---:|
| 사육장 | 3 |
| 환경 프리셋 | 3 |
| 생물 역할군 | 6 |
| 식물 역할군 | 6 |
| 바닥재 | 4 |
| 구조물 | 4 |
| 환경 장비 | 5 |
| 먹이 범주 | 4 |
| 시나리오 | 4 |

## 품질 우선순위

1. 경고 원인과 해결 방법의 명료성
2. 생물 행동과 상태의 연결
3. 합사 위험의 예측 가능성
4. 여러 사육장 전환의 편의성
5. 도트 애니메이션 일관성
6. 배경화면 모드 안정성
7. 콘텐츠 수량

콘텐츠 수량 때문에 핵심 행동이나 UI 품질을 낮추지 않습니다.

## 축소 기준

일정이 부족할 경우 다음 순서로 축소합니다.

1. 식물과 구조물 종류 감소
2. 시나리오 수 감소
3. 개구리형과 도마뱀형 생물 후순위 이동
4. 복잡한 먹이 경쟁 연출 단순화
5. 배경화면 모드의 클릭 인터랙션 축소

다음은 축소하지 않습니다.

- 사육장 간 이동
- 환경 적합도
- 합사 진단
- 포식 위험 경고
- 달팽이형·등각류형 세로형 프로토타입
- 기본 관찰 모드
