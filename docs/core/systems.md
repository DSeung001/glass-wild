# 핵심 시스템 규칙

## 목적

이 문서는 Glass Wild의 시뮬레이션을 구성하는 공통 규칙을 정의합니다. 실제 수치보다 시스템 간 인과관계와 판정 순서를 우선합니다.

## 시스템 계층

| 계층 | 주요 요소 | 대표 문제 |
|---|---|---|
| 환경 | 온도, 습도, 청결도, 먹이 상태 | 환경 부적합, 오염, 굶주림 |
| 공간 | 수용량, 활동 구역, 은신처, 식물 밀도 | 과밀, 스트레스, 영역 충돌 |
| 관계 | 포식, 경쟁, 사회성, 동종 갈등 | 추적, 공격, 먹이 독점 |
| 상태 | 건강, 배고픔, 스트레스, 부상 | 행동 변화, 활동 감소 |
| 운영 | 시간 속도, 경고, 격리, 이동 | 우선순위 판단, 복구 |

## 핵심 환경 수치

| 수치 | 범위 | 기본 표시 | 주요 영향 |
|---|---:|---|---|
| 온도 | 게임 내 단위 또는 °C | 현재값과 권장 범위 | 활동량, 스트레스, 수분 손실 |
| 습도 | 0~100 | 백분율 | 활동량, 식물 상태, 곰팡이 위험 |
| 청결도 | 0~100 | 점수 | 건강, 스트레스, 질병 위험 대체 |
| 먹이 충족도 | 생물별 0~100 | 상태 또는 게이지 | 배고픔, 경쟁, 포식 행동 |

## 보조 환경 조건

보조 조건은 숫자 대신 상태 태그로 표시합니다.

- 바닥재 적합성: 적합 / 허용 / 부적합
- 환기: 부족 / 적정 / 과다
- 은신처: 부족 / 적정 / 충분
- 식물 밀도: 낮음 / 적정 / 과밀
- 수용 밀도: 여유 / 적정 / 과밀
- 조명: 부족 / 적정 / 과다
- 수역: 없음 / 얕음 / 있음 (논리 `water` 셀·수위, D-009)

## 생물별 환경 적합도

환경 적합도는 사육장 전체가 아니라 각 생물마다 계산합니다.

### 1차 계산 요소

| 요소 | 가중치 초안 |
|---|---:|
| 온도 | 25 |
| 습도 | 25 |
| 바닥재 | 15 |
| 공간과 밀도 | 15 |
| 은신처 | 10 |
| 조명·수역 등 종별 조건 | 10 |

가중치는 플레이테스트 후 조정합니다.

### 상태 구간

| 점수 | 상태 | 효과 |
|---:|---|---|
| 80~100 | 안정 | 정상 행동, 스트레스 감소 |
| 60~79 | 주의 | 활동 감소, 정보 제공 |
| 40~59 | 위험 | 건강 저하, 먹이 거부, 공격성 증가 가능 |
| 0~39 | 긴급 | 탈진 또는 부상 가능, 자동 감속 |

## 사육장 전체 안정도

전체 안정도는 단순 평균이 아닙니다.

1. 가장 낮은 생물 적합도를 기준값으로 사용합니다.
2. 포식 위험, 긴급 경고, 과밀에 감점을 적용합니다.
3. 청결도와 먹이 충족도를 보정값으로 반영합니다.
4. 위험 요소가 하나라도 긴급이면 전체 상태도 긴급으로 표시합니다.

## 생물 상태

| 상태 | 범위 또는 단계 | 표현 |
|---|---|---|
| 건강 | 정상 / 회복 중 / 부상 / 탈진 | 행동 속도와 자세 변화 |
| 배고픔 | 포만 / 보통 / 배고픔 / 극심 | 탐색 행동 빈도 증가 |
| 스트레스 | 낮음 / 보통 / 높음 / 극심 | 은신, 회피, 위협 행동 |
| 사회 상태 | 안정 / 경쟁 / 영역 충돌 / 포식 위험 | 관계 아이콘과 진단 문구 |

## 시간 진행

- 일반 플레이 기본 속도: 1배
- 지원 속도: 일시정지, 1배, 2배, 4배
- 긴급 상황: 0.25배 또는 자동 일시정지
- 배치·진단 화면: 사용자 설정에 따라 자동 일시정지
- 오프라인 진행: MVP 제외

## 경고 단계

| 단계 | 의미 | 자동 동작 |
|---|---|---|
| 정보 | 변화가 시작됨 | 없음 |
| 주의 | 조치 권장 | 대시보드 표시 |
| 위험 | 빠른 조치 필요 | 대상 강조 |
| 긴급 | 사고 진행 또는 생존 범위 이탈 | 자동 감속, 긴급 행동 표시 |

모든 경고는 문제, 대상, 원인, 권장 조치를 포함해야 합니다.

## 복구 원칙

- MVP에서는 즉시 게임 오버를 사용하지 않습니다.
- 사고는 부상, 탈진, 자동 격리로 연결합니다.
- 조치 후 수치와 행동은 즉시 정상화되지 않고 점진적으로 회복합니다.
- 복구 후 원인과 재발 방지 방법을 요약합니다.

## 저장 대상

- 사육장별 프리셋과 환경 수치
- 배치된 생물, 식물, 구조물, 장비
- 생물별 건강, 배고픔, 스트레스, 현재 행동
- 경고와 사고 상태
- 게임 속도와 배경화면 모드 설정
