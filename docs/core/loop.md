# 핵심 게임 루프

## 루프 개요

Glass Wild는 세 개의 중첩된 루프로 구성됩니다.

| 루프 | 시간 규모 | 목적 |
|---|---|---|
| 순간 행동 루프 | 수초~1분 | 먹이 공급, 청소, 장비 조절, 생물 이동 |
| 사육장 안정화 루프 | 3~10분 | 한 사육장의 환경과 합사 상태를 안정화 |
| 다중 사육장 운영 루프 | 10분 이상 | 세 사육장의 위험 우선순위를 관리하고 조합을 재설계 |

배경화면 모드는 별도 진행 루프가 아니라 이 상태를 낮은 인터랙션으로 관찰하는 표현 모드입니다.

---

## 1. 일반 플레이 기본 루프

1. 사육장을 선택한다.
2. 배치할 생물을 선택한다.
3. 생물의 권장 환경과 기존 생물 관계를 확인한다.
4. 바닥재, 식물, 은신처와 장비를 구성한다.
5. 생물을 배치한다.
6. 환경 적합도와 합사 판정을 확인한다.
7. 생물 행동과 환경 변화를 관찰한다.
8. 경고 또는 이상 행동의 원인을 진단한다.
9. 먹이 공급, 청소, 환경 조절, 이동 또는 격리로 대응한다.
10. 안정 상태를 일정 시간 유지한다.

## 루프 완료 조건

한 사육장이 다음 조건을 동시에 만족하면 `안정화 완료`로 판단합니다.

- 전체 안정도 80 이상
- 긴급 경고 0개
- 모든 생물의 건강 상태가 `정상` 또는 `회복 중`
- 명백한 포식 관계 없음
- 청결도 70 이상
- 위 조건을 180초 동안 유지

안정화 완료는 콘텐츠 해금이나 재화를 주지 않습니다. 대신 다음 피드백을 제공합니다.

- 사육장 상태 배지
- 안정 유지 시간
- 주요 성공 요인 요약
- 선택적으로 다음 실험 조합 추천

## 2. 순간 행동 루프

### 관찰

플레이어는 대시보드, 사육장 화면 또는 생물 행동을 통해 문제를 발견합니다.

### 진단

대상을 선택하면 다음 정보가 표시됩니다.

- 현재 행동
- 핵심 상태
- 가장 큰 부정 요인
- 보조 요인
- 추천 조치

### 행동

| 문제 | 가능한 조치 |
|---|---|
| 습도 부족 | 분무 장치 작동, 수분 보유 바닥재 추가, 환기 감소 |
| 습도 과다 | 환기 장치 작동, 분무 중지, 바닥재 교체 |
| 온도 부족 | 가열 장치 작동, 따뜻한 구역 제공 |
| 온도 과다 | 가열 중지, 환기, 식물 또는 그늘 구조물 추가 |
| 청결도 부족 | 오염물 제거, 남은 먹이 회수, 청소 생물 검토 |
| 배고픔 | 적합한 먹이 공급 |
| 먹이 경쟁 | 먹이 위치 분산, 개체 분리 |
| 은신처 부족 | 구조물 또는 식물 추가 |
| 과밀 | 개체를 다른 사육장으로 이동 |
| 포식 위험 | 즉시 분리 또는 임시 격리 |

### 결과 확인

행동 직후 예상 효과를 표시하고, 실제 수치와 행동 변화는 시간에 따라 반영합니다.

## 3. 다중 사육장 운영 루프

### 전체 상태 확인

대시보드에서 세 사육장을 다음 우선순위로 정렬할 수 있습니다.

1. 긴급 사고 진행 중
2. 포식 위험
3. 환경 생존 범위 이탈
4. 부상 또는 탈진
5. 먹이 부족
6. 과밀 또는 경쟁
7. 경미한 환경 부적합
8. 안정

### 사육장 간 이동

생물 이동은 핵심 전략 행동입니다.

이동 전 다음을 비교합니다.

- 출발 사육장의 상태 개선 예상
- 도착 사육장의 환경 적합도
- 도착 사육장의 남은 수용량
- 도착 사육장의 기존 생물 관계
- 이동 후 양쪽 사육장의 예상 안정도

### 재설계

플레이어는 다음 방식으로 사육장 역할을 변경할 수 있습니다.

- 프리셋 변경
- 장비 조합 변경
- 바닥재 교체
- 식물과 은신처 재배치
- 생물군 재구성

## 4. 배경화면 모드 루프

### 진입 조건

- 사용자가 직접 배경화면 모드를 선택
- 또는 일반 창을 최소화할 때 자동 전환하도록 설정

자동 전환 여부는 사용자 설정으로 둡니다.

### 평상시 동작

- 일반 플레이와 동일한 사육장 데이터를 사용
- 기본 1배속 진행
- 도트 애니메이션 지속
- UI, 수치, 메뉴 최소화
- 마우스 입력은 기본적으로 통과시키거나 비활성 상태 유지
- 클릭 또는 지정 단축키로 인터랙션 모드 전환

### 배경화면 모드의 최소 정보

| 상태 | 표현 방식 |
|---|---|
| 안정 | 별도 UI 없음, 자연 행동 유지 |
| 정보 | 작은 비점멸 아이콘 |
| 주의 | 해당 사육장 모서리에 상태 아이콘 |
| 긴급 | 자동 감속, 짧은 배너, 문제 대상 표시 |

### 배경화면 모드의 제한

- 배치 가꾸기 불가
- 세부 데이터 패널 비노출
- 복잡한 드래그 조작 불가
- 긴급 조치는 인터랙션 모드 전환 후 수행

### 관찰 이벤트

배경화면 모드에서는 낮은 빈도의 작은 사건을 보여줍니다.

- 생물이 먹이를 발견함
- 식물 아래에서 휴식함
- 두 생물이 서로 피하거나 지나감
- 청소 생물이 남은 먹이 주변에 모임
- 야행성 생물의 활동량이 변화함

이벤트는 보상을 제공하지 않으며 관찰 재미를 위한 연출입니다.

## 5. 경고에서 복구까지의 루프

1. 이상 징후가 행동으로 나타난다.
2. 정보 또는 주의 알림이 발생한다.
3. 문제가 지속되면 위험 상태로 상승한다.
4. 사고 직전 게임이 자동 감속한다.
5. 플레이어가 원인을 확인한다.
6. 조치를 선택한다.
7. 수치와 행동이 점진적으로 회복된다.
8. 복구 요약이 표시된다.

## 복구 요약 항목

- 발생한 문제
- 영향을 받은 생물
- 문제 지속 시간
- 플레이어가 수행한 조치
- 상태가 개선된 이유
- 재발 방지 권장 사항

## 6. 시나리오 모드 루프

모든 콘텐츠는 열려 있지만 시나리오가 학습 목표를 제공합니다.

| 시나리오 | 핵심 학습 | 완료 조건 |
|---|---|---|
| 첫 안정 사육장 | 환경 기초 | 달팽이와 등각류를 3분간 안정 유지 |
| 서로 다른 환경 | 프리셋과 장비 | 고습·건조 사육장 안정도 각각 80 이상 |
| 위험한 합사 | 포식 경고 | 사고 전에 포식자와 피식자 분리 |
| 세 사육장 운영 | 우선순위 관리 | 5분간 모든 사육장 긴급 상태 없음 |

---

## 7. 시간 속도

| 속도 | 사용 목적 |
|---|---|
| 일시정지 | 배치, 진단, 긴급 조치 |
| 1배속 | 기본 플레이와 배경화면 모드 |
| 2배속 | 환경 변화 관찰 |
| 4배속 | 장기 안정성 시험 |
| 0.25배속 | 긴급 상황 자동 감속 |

오프라인 시간 진행은 MVP에서 제외합니다.
