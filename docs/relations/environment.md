# 환경 상호작용 규칙

## 목적

사육장 구성 요소가 온도, 습도, 청결도와 공간 조건에 미치는 영향을 일관된 방식으로 정의합니다.

## 환경 변화 모델

환경 수치는 즉시 목표값으로 바뀌지 않습니다.

1. 사육장 프리셋이 기본 변화율을 제공합니다.
2. 바닥재, 식물과 장비가 변화율을 수정합니다.
3. 생물 수, 먹이와 오염물이 부하를 추가합니다.
4. 현재값이 시간에 따라 목표 범위로 이동합니다.

## 생물 부하

생물은 다음 방식으로 환경에 영향을 줍니다.

- 개체 수 증가: 청결도 감소 속도 증가
- 큰 생물: 공간과 먹이 부하 증가
- 군집 생물: 과밀 임계치에 빠르게 접근
- 청소 생물: 특정 부패물의 청결도 피해 완화
- 초식 생물: 식물 효과 감소 가능
- 포식자: 먹이 생물 수와 행동에 영향

## 환경 구역

사육장 전체에 단일 값만 사용하면 배치의 의미가 약해질 수 있습니다. MVP 세로형 프로토타입에서 다음 두 방식을 비교합니다.

### 방식 A: 사육장 단일 수치

- 구현이 단순함
- UI가 명확함
- 배치 위치의 의미가 약함

### 방식 B: 간단한 구역 수치

- 왼쪽·중앙·오른쪽 또는 바닥·상부 구역으로 분할
- 가열 장치, 수역과 은신처의 위치가 의미를 가짐
- 구현과 진단 UI가 복잡해짐

MVP 기본 권장은 단일 수치이며, 가열 구역과 수역만 태그형 국소 구역으로 처리하는 방식입니다.

## 적합도 판정 원칙

- 권장 범위 내부: 감점 없음
- 권장 범위에서 조금 벗어남: 점진적 감점
- 생존 가능 범위 이탈: 큰 감점과 위험 경고
- 여러 조건이 동시에 나쁘면 합산하되 한 조건의 긴급 상태를 평균으로 숨기지 않음

## 환경 변화 피드백

플레이어 행동 전:

- 예상 방향 표시
- 장점과 부작용 표시
- 영향을 받는 생물 표시

플레이어 행동 후:

- 실제 수치 변화
- 생물 행동 변화
- 경고 단계 변화
- 목표 안정도까지 예상 추세

## 프리셋 규칙

프리셋은 사육장의 고정 타입이 아니라 다음 설정 묶음입니다.

- 초기 온도와 습도
- 기본 변화율
- 기본 바닥재
- 기본 장비 상태
- 권장 식물 조합

프리셋을 변경해도 생물과 배치물이 자동 삭제되지 않습니다. 변경 전에 예상 위험을 보여줍니다.
