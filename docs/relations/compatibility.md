# 생물 합사와 관계 판정

## 목적

두 개 이상의 생물을 같은 사육장에 배치할 때 안전, 경쟁, 영역 충돌과 포식 위험을 일관된 순서로 판정합니다.

## 판정 순서

1. 현재 사육장의 환경 적합도
2. 남은 수용량과 활동 구역 충돌
3. 먹이 경쟁
4. 사회성과 영역성
5. 포식 관계
6. 현재 배고픔과 스트레스에 따른 위험 보정

## 최종 판정

| 결과 | 의미 | 배치 처리 |
|---|---|---|
| 안전 | 의미 있는 충돌 없음 | 바로 배치 가능 |
| 주의 | 경쟁, 과밀 또는 조건부 공격 가능 | 경고 후 배치 가능 |
| 위험 | 명확한 포식 또는 치명적 환경 충돌 | 강한 경고 후 배치 가능 |
| 긴급 | 이미 추적·공격·탈진이 진행 중 | 자동 감속과 즉시 행동 제공 |

위험한 배치를 완전히 차단하지 않습니다. 단, 플레이어가 결과를 예상할 수 있도록 배치 전 정보를 제공합니다.

## 포식 판정 요소

| 요소 | 판정 영향 |
|---|---|
| 포식자 태그와 대상 태그 일치 | 명확한 위험 증가 |
| 크기 차이 | 대상이 작을수록 위험 증가 |
| 활동 구역 중첩 | 만남 빈도 증가 |
| 포식자의 배고픔 | 추적 확률 증가 |
| 은신처 부족 | 피식자의 회피 가능성 감소 |
| 포식자의 스트레스 | 공격 행동 증가 가능 |

MVP에서는 복잡한 확률 모델 대신 다음 세 단계 관계를 사용합니다.

- 명백한 포식 관계: 위험
- 조건부 공격 관계: 주의
- 직접적인 공격 관계 없음: 안전

## 경쟁 판정

| 경쟁 유형 | 발생 조건 | 결과 |
|---|---|---|
| 먹이 경쟁 | 동일 먹이, 공급량 부족 | 약한 개체의 배고픔 증가 |
| 은신처 경쟁 | 요구량보다 은신처 부족 | 스트레스와 회피 증가 |
| 활동 공간 경쟁 | 동일 활동 구역 과밀 | 이동 충돌과 영역 행동 증가 |
| 동종 경쟁 | 단독 성향 개체 복수 배치 | 위협 또는 공격 가능 |

## 과밀 판정

사육장 수용량과 생물별 공간 점유를 사용합니다.

- 총 점유량 80% 미만: 여유
- 총 점유량 80~100%: 적정 또는 주의
- 총 점유량 100% 초과: 과밀
- 영역성 생물은 별도 공간 보정치를 적용

정확한 임계치는 플레이테스트 후 조정합니다.

## 사고 진행 단계

1. 포식자가 대상에게 관심을 보임
2. 시선 고정 또는 접근 행동
3. 추적 경고 발생
4. 게임 자동 감속
5. 플레이어 대응 시간 제공
6. 미대응 시 공격과 부상
7. 대상 자동 격리 또는 긴급 조치 요구

## 배치 전 UI 필수 정보

- 현재 환경 적합도
- 남은 수용량
- 기존 생물별 관계
- 가장 위험한 관계 한 줄 요약
- 예상 안정도 변화
- 다른 사육장 추천
