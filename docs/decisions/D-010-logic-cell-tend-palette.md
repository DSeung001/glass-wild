# D-010 논리 셀 크기와 가꾸기 팔레트

- 우선순위: P0
- 상태: Decided
- 관련 문서: `docs/decisions/D-003-placement-model.md`, `docs/decisions/D-009-habitat-surfaces-water.md`, `docs/core/glossary.md`, `docs/ui/layout.md`, `docs/ui/flow.md`

## 최종 결정

- 상태: Decided
- 결정:
  1. **논리 셀** 한 변은 **1cm × 1cm**. 측면 뷰에서는 사육장 가로×높이 평면에 논리 격자를 깐다. 깊이(cm)는 규격 값으로만 두고 측면 격자에는 펼치지 않는다.
  2. 가꾸기 팔레트에서 종류를 고른 뒤 배치한다.
     - **동물**: 클릭 1회 → 해당 논리 셀에 스냅. UI에는 **표시 이름**(`display_name`), 시각은 더미 사각형
     - **지형** (벽지·땅·물): 두 모드
       - **점**: 클릭 1회 → 논리 셀 1칸
       - **사각형**: 드래그로 축정렬 사각형 영역을 일괄 채움 (브러시 연속 칠하기 아님)
     - **식물·이끼**: 클릭 1회. `plant_surfaces`에 허용된 면(`floor_*` / `wall_*`)에만 설치. 이끼도 동일 규칙(식재 오버레이)
  3. 가꾸기 화면의 **표시 격자**는 **5cm마다** 선을 그린다 (논리 셀 5칸). 스냅·저장·판정 단위는 항상 논리 셀(1cm)이다. 표시 격자는 **ON/OFF** 할 수 있다 (기본 ON).
- 결정일: 2026-07-19
- 결정자: 프로젝트 책임자
- 이유: 물리 규격(cm)과 가꾸기 조작을 맞추고, 지형은 점·사각형으로 면적을 다루며 생물/식물은 점 배치로 둔다. 식물·이끼는 D-009 설치 면과 일치시킨다. 표시 격자 5cm는 가독성, 1cm 스냅은 정밀도용이다.
- 재검토 조건: 1cm가 과밀·성능·조작에 부담이거나, 표시 격자 간격이 혼란을 줄 때.

## 측면 셀 수 예

| 규격 (W×D×H cm) | 측면 논리 셀 (W×H) |
|---|---|
| 30×30×45 | 30 × 45 |
| 45×45×45 | 45 × 45 |
| 60×45×45 | 60 × 45 |
| 60×45×60 | 60 × 60 |
| 90×45×60 | 90 × 60 |
| 120×60×60 | 120 × 60 |

용어는 [`docs/core/glossary.md`](../core/glossary.md)의 **논리 셀** / **표시 격자** / **스프라이트 셀**을 따른다.
