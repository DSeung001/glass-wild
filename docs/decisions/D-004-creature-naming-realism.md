# D-004 실제 종명과 현실성 수준

- 우선순위: P1
- 상태: Pending
- 관련 문서: `docs/entities/animals.md`, `docs/relations/compatibility.md`

## 결정해야 하는 질문

1. 실제 생물 종명을 사용할 것인가?
2. 온도, 습도와 합사 정보를 실제 사육 기준에 맞출 것인가?
3. 게임 규칙을 위해 현실 정보를 단순화할 때 어떻게 표시할 것인가?

## 선택지

### A. 실제 종명과 실제 사육 기준

장점:

- 교육적 가치와 몰입 증가
- 실제 테라리움 사용자에게 설득력 있음

위험:

- 정보 검수 비용이 큼
- 잘못된 사육 정보가 실제 동물 복지 문제로 이어질 수 있음
- 종별 예외가 많아 게임 규칙이 복잡해짐

### B. 가상 이름과 역할형 생물

장점:

- 명확한 게임 규칙 설계 가능
- 밸런스와 콘텐츠 제작이 쉬움

위험:

- 현실 테라리움 감성이 약해질 수 있음
- 생물 애호가에게 매력이 낮을 수 있음

### C. 실제 생물에서 영감을 받은 가상 종

- 실루엣과 행동은 실제 생물을 참고
- 이름과 수치는 게임용으로 명확하게 구분
- 도감에 실제 사육 지침이 아님을 표시

## 권장안

MVP는 C를 권장합니다.

- 역할형 가상 이름으로 게임 규칙 검증
- 실제 생물 정보를 직접적인 사육 지침처럼 제공하지 않음
- 정식 버전에서 전문가 또는 신뢰할 수 있는 자료 검수를 거쳐 실제 종 모드 검토

## 필수 표시 원칙

- 게임 수치가 실제 사육 지침이 아님을 명시
- 실제 생물명 사용 시 종 단위로 검수
- 포식과 합사 정보는 안전을 우선해 보수적으로 표현
- 현실과 다른 단순화는 도감에서 설명

## 영향

- 생물 이름과 아트 방향
- 도감 문구
- 관계표와 환경 수치
- 외부 검수 일정
- 스토어 설명과 면책 문구
