# D-007 MVP 콘텐츠 수량 고정 여부

- 우선순위: P2
- 상태: Pending
- 관련 문서: `docs/mvp/scope.md`, `docs/mvp/vertical-slice.md`

## 결정해야 하는 질문

세로형 프로토타입 이후에도 생물 6종, 식물 6종과 아이템 전체 수량을 첫 MVP에 유지할지 결정합니다.

## 선택지

### A. 계획 수량 유지

- 생물 6종, 식물 6종과 전체 아이템을 포함
- 다양한 환경과 합사 규칙을 보여줄 수 있음
- 아트와 애니메이션 제작량이 큼

### B. 생물 4종 중심 MVP

- 달팽이형, 등각류형, 귀뚜라미형, 사마귀형만 포함
- 환경 기초와 포식 관계까지 검증 가능
- 고습 포식자와 건조 상위 생물의 복합 설계는 후속으로 이동

### C. 세로형 프로토타입 결과에 따라 동적 결정

- 애니메이션 제작 시간, UI 복잡도와 플레이테스트 결과를 기준으로 4종 또는 6종 선택
- 일정 관리에 유리
- 초기 홍보 콘텐츠 수량을 확정하기 어려움

## 권장안

C를 권장합니다.

결정 시점:

- 달팽이형·등각류형 세로형 프로토타입 완료
- 귀뚜라미형·사마귀형 관계 시스템 검증 완료

6종 유지 조건:

- 생물 1종당 애니메이션 제작 시간이 예산 내에 있음
- 관계 규칙과 UI가 추가 종을 수용할 수 있음
- 핵심 시스템의 오류 수정 일정이 안정됨

4종 축소 조건:

- 에셋 일관성 유지가 어려움
- 배경화면 성능 문제가 있음
- 생물 추가가 새로운 플레이 선택을 만들지 못함

## 영향

- 관계표 규모
- 시나리오 수
- 배경화면 성능
