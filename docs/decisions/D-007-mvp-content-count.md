# D-007 MVP 콘텐츠 수량 고정 여부

- 우선순위: P2
- 상태: Decided
- 관련 문서: `docs/mvp/scope.md`, `docs/mvp/vertical-slice.md`

## 최종 결정

- 상태: Decided
- 결정: MVP의 **첫 콘텐츠 축·예시**는 팔루다리움(열대기후)이며, 아래 목록으로 환경·합사·관찰을 먼저 검증한다. 이 목록은 닫힌 최종 로스터가 아니다. 동물군·식물군은 이후 축(온대·건조 등)과 역할 확장으로 **추가할 수 있다**. 세로형 프로토타입 이후 제작량·성능에 따라 일부 후순위 이동은 가능하나, MVP 첫 축의 목표 예시는 이 구성을 따른다.

동물 (첫 축 예시):

- 네리트 달팽이
- 체리새우
- 등각류
- 모어닝게코
- 스프링테일
- 다트프록
- 앤들러 구피
- 파이어밸리 뉴트

종별 접근 영역(바닥·벽·수역)은 [`docs/entities/animals.md`](../entities/animals.md) locomotion 표를 따른다.

식물 (첫 축 예시):

- 포토스
- 피쿠스 푸밀라
- 피토니아
- 셀라기넬라

이끼 (첫 축 예시):

- 깃털이끼
- 솔이끼
- 비단이끼

- 결정일: 2026-07-19
- 결정자: 프로젝트 책임자
- 이유: 열대 팔루다리움 한 축으로 환경·합사·관찰 재미를 먼저 고정한다. 전체 동물군을 이 목록으로 한정하지 않는다.
- 재검토 조건: 세로형·관계 시스템 검증 후 아트 제작량 또는 성능이 목표를 초과할 때. 추가 동물군·기후 축을 넣을 때.
