## 최종 결정

- 상태: Decided
- 선택한 완료 기준: 핵심 흐름 프로토타입 + 기술 스파이크 후, 세로형 프로토타입에 필요한 화면만 고해상도 확정. MVP 전체 고해상도 디자인은 세로형 프로토타입 테스트 이후 확장.
- Godot 시작 조건:
  - 대시보드, 개별 사육장, 가꾸기, 배치 진단, 긴급, 배경화면 화면의 와이어프레임 완료
  - 핵심 사용자 흐름 5개 연결
  - 상태 컴포넌트와 용어 확정
  - 배경화면 기술 제약 확인
  - 도트 기준 셀 크기 확인
  - 세로형 프로토타입 화면의 고해상도 디자인 완료
- 결정일: 2026-07-19
- 결정자: 프로젝트 책임자
- 이유: 기술 제약을 반영하기 전에 전체 고해상도를 만들지 않고, 검증된 범위만 디자인해 재작업을 줄인다.
- 재검토 조건: 세로형 프로토타입에서 UI 재설계가 과도하거나, 임시 와이어프레임이 장기간 남을 때.
