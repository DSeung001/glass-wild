# 사용자 의사결정 목록

## 목적

이 폴더에는 구현 방향, 일정과 콘텐츠 범위를 바꿀 수 있어 프로젝트 책임자가 직접 확정한 결정을 둡니다. 각 문서는 최종 결정만 유지합니다.

## 상태 규칙

| 상태 | 의미 |
|---|---|
| Decided | 최종 결정 완료 |
| Revisit | 구현 또는 테스트 결과로 재검토 필요 |
| Superseded | 다른 결정으로 대체됨 |
