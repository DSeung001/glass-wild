# D-006 배경화면 입력과 알림 방식

- 우선순위: P0
- 상태: Decided
- 관련 문서: `docs/technical/wallpaper-spike.md`, `docs/ui/layout.md`

## 최종 결정

- 상태: Decided
- 기본 입력 방식: 마우스 입력 통과
- 인터랙션 진입 방식: 사용자 지정 전역 단축키 또는 트레이 메뉴. 인터랙션 중에는 생물과 최소 UI 클릭 가능. 종료는 빈 영역 클릭, ESC 또는 일정 시간 후 자동 복귀.
- 긴급 알림 방식: 정보·주의는 작은 비점멸 아이콘, 위험은 문제 사육장·대상 강조, 긴급은 자동 감속과 짧은 배너. 일반 화면으로 자동 전환하지 않음.
- 운영체제 알림 기본값: 비활성 (사용자가 선택 시 활성)
- 결정일: 2026-07-19
- 결정자: 프로젝트 책임자
- 이유: 다른 앱 사용을 방해하지 않으면서도, 단축키·트레이로 필요 시 개입하고 긴급 신호를 놓치지 않게 한다.
- 재검토 조건: 인터랙션 진입 발견성이 낮거나, 알림 피로·다중 모니터 불일치가 플레이테스트에서 드러날 때.
