# D-003 사육장 배치 방식

- 우선순위: P0
- 상태: Pending
- 관련 문서: `docs/entities/items.md`, `docs/ui/layout.md`

## 결정해야 하는 질문

식물, 구조물, 장비와 먹이를 자유 좌표에 놓을지, 격자에 맞춰 놓을지 결정해야 합니다.

## 선택지

### A. 완전 자유 배치

장점:

- 꾸미기 자유도가 높음
- 자연스러운 테라리움 표현 가능

위험:

- 충돌, 길찾기와 저장이 복잡함
- 작은 도트 오브젝트 선택이 어려움
- 배치 위치에 따른 환경 효과 계산이 복잡함

### B. 완전 격자 배치

장점:

- 구현과 저장이 단순함
- 충돌과 수용량 계산이 명확함
- 마우스 조작이 안정적임

위험:

- 인공적으로 보일 수 있음
- 도트 테라리움의 자연스러운 구성 제한

### C. 논리 격자 + 시각적 자유 배치

- 내부 판정은 격자 또는 슬롯을 사용
- 렌더링 위치에 작은 오프셋을 허용
- 장식은 더 자유롭게 배치하고 기능 아이템은 격자에 맞춤

## 권장안

C를 권장합니다.

- 환경 효과와 길찾기는 논리 격자 기준
- 식물과 구조물은 셀 내부 시각 오프셋 허용
- 먹이는 자유롭게 보이되 가장 가까운 논리 셀에 귀속
- 편집 화면에서는 격자를 표시하고 관찰 화면에서는 숨김

## 판단 기준

- 마우스로 쉽게 선택하고 이동할 수 있는가
- 생물 이동 경로가 예측 가능한가
- 도트 화면이 지나치게 격자처럼 보이지 않는가
- 저장 데이터가 단순한가
- 아이템 환경 효과 범위를 명확히 표시할 수 있는가

## 영향

- Figma 편집 모드
- Godot 좌표와 길찾기 구조
- 저장 형식
- 환경 구역 계산
- 에셋 피벗과 크기
