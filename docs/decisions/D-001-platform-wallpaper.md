## 최종 결정

- 상태: Decided
- 결정: Windows 우선. Phase 2 기술 스파이크 통과 시 배경화면 모드를 MVP 필수로 확정. 실패 시 일반 무테 관찰 모드를 MVP 대체안으로 사용. macOS와 Linux는 MVP 이후 검토.
- 결정일: 2026-07-19
- 결정자: 프로젝트 책임자
- 이유: 제품 차별점인 배경화면은 유지하되, 투명 창 호환성 위험이 전체 일정을 막지 않도록 스파이크 결과에 따라 범위를 조정한다.
- 재검토 조건: 배경화면 스파이크가 실패하거나, 무테 관찰 모드만으로는 핵심 질문이 충족되지 않을 때.
