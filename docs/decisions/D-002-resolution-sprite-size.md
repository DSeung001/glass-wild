# D-002 기본 해상도와 도트 셀 크기

- 우선순위: P0
- 상태: Decided
- 관련 문서: `docs/design/figma-plan.md`, `docs/art/pixel-art-pipeline.md`

## 최종 결정

- 상태: Decided
- 기준 해상도: 1920×1080
- 최소 지원 해상도: 1280×720 (정보 손실 없이 동작)
- 기본 생물 셀: 기술 스파이크에서 32×32·48×48 달팽이형을 비교한 뒤 기본 셀을 확정. 생물 체급(소·중·대)에 따라 32·48·64 셀을 지원하며, 도마뱀형 등은 최대 64×64까지 사용 가능.
- UI 스케일 규칙: 가변 레이아웃, 화면 확대는 정수 배율 우선
- 결정일: 2026-07-19
- 결정자: 프로젝트 책임자
- 이유: 데스크톱 기준 해상도와 가변 UI로 Figma·Godot 명세를 맞추고, 작은 생물부터 큰 생물까지 읽을 수 있는 체급별 셀을 허용한다.
- 재검토 조건: 스파이크에서 배경화면 가독성·사육장 수용량·제작량이 기준을 만족하지 못할 때.
