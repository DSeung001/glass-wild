# D-005 사망·부상·실패 처리

- 우선순위: P1
- 상태: Decided
- 관련 문서: `docs/core/systems.md`, `docs/mvp/scope.md`

## 최종 결정

- 상태: Decided
- 결정: MVP에서 생물 사망 없음. 스트레스·환경 악화는 위험 경고 → 탈진 또는 부상 → 자동 일시정지 → 임시 격리 → 환경 수정 → 회복 순으로 처리한다. 회복이 필요하면 병원(치료·격리 공간)으로 이동한다. 포식 사고는 사실적 묘사 없이 짧은 공격 동작, 부상, 분리 조치로 표현한다. 게임 오버 없음. 정식 버전에서 사망 옵션은 플레이테스트 후 재검토.
- 결정일: 2026-07-19
- 결정자: 프로젝트 책임자
- 이유: 복구 플레이와 잔잔한 관찰 톤을 유지하면서도 위험의 결과를 부상·격리·병원 이동으로 명확히 한다.
- 재검토 조건: 정식 버전 톤 테스트에서 사망 옵션 수요가 명확하거나, 병원 흐름이 과도하게 복잡해질 때.
