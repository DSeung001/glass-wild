# 화면별 UI 레이아웃

## 목표

UI는 환경 진단에는 명확하고, 생태계 관찰에는 방해가 적어야 합니다. 모든 상태는 색상뿐 아니라 아이콘과 텍스트로 함께 표시합니다.

## 화면 목록

| ID | 화면 | 주요 목적 |
|---|---|---|
| UI-01 | 시작 화면 | 계속하기, 새 게임, 설정, 배경화면 진입 |
| UI-02 | 사육장 대시보드 | 세 사육장의 위험 우선순위 확인 |
| UI-03 | 개별 사육장 관찰 | 생물 행동과 환경 상태 관찰 |
| UI-04 | 사육장 편집 | 식물, 장비, 구조물과 생물 배치 |
| UI-05 | 생물 목록 | 생물 탐색, 필터, 비교 |
| UI-06 | 배치 전 진단 | 환경과 합사 위험 확인 |
| UI-07 | 대상 상세 패널 | 생물·식물·장비 상태와 원인 확인 |
| UI-08 | 긴급 오버레이 | 사고 원인과 즉시 행동 제공 |
| UI-09 | 배경화면 모드 | UI 최소화 관찰 |
| UI-10 | 설정 | 화면, 시간, 알림, 배경화면 동작 설정 |

## UI-01 시작 화면

필수 항목:

- 계속하기
- 새 생태계
- 배경화면 모드 시작
- 설정
- 종료
- 마지막 저장 시각과 대표 사육장 상태

## UI-02 전체 사육장 대시보드

### 상단 요약

- 전체 사육장 수
- 안정, 주의, 위험, 긴급 개수
- 현재 게임 속도
- 전체 일시정지
- 배경화면 모드 전환

### 사육장 카드

- 사육장 이름
- 환경 프리셋
- 대표 도트 미리보기
- 전체 안정도
- 생물 수와 수용량
- 가장 심각한 문제
- 마지막 사건
- 상태 배지

사육장 카드는 긴급, 위험, 주의, 안정 순으로 정렬할 수 있습니다.

## UI-03 개별 사육장 관찰

### 상단 바

- 사육장 이름
- 전체 안정도
- 경고 수
- 일시정지, 1배, 2배, 4배
- 편집 모드 전환

### 왼쪽 패널

- 사육장 A, B, C 전환
- 각 사육장의 상태와 핵심 문제

### 중앙 관찰 영역

- 생물, 식물, 먹이, 오염물과 구조물
- 선택 대상 강조
- 필요할 때만 이동 목표와 관계선 표시
- 오버레이: 습도, 온도, 오염, 활동 구역, 관계

### 오른쪽 패널

선택 없음:

- 전체 환경 상태
- 주요 문제
- 추천 조치
- 생물 조합 안전도

생물 선택:

- 건강, 배고픔, 스트레스, 적합도
- 현재 행동과 원인
- 위험 관계
- 이동·격리 버튼

식물 또는 장비 선택:

- 상태
- 환경 효과
- 작동 상태
- 예상 변화

### 하단 메뉴

- 환경
- 바닥재
- 식물
- 생물
- 먹이
- 장비
- 청소
- 이동

## UI-04 편집 모드

- 관찰 모드와 시각적으로 구분
- **표시 격자** 표시 (논리 셀 0.5cm 스냅, D-010). 관찰에서는 숨김
- 측면 팔레트: 지형 / 식물·이끼 / 동물
- 배치 제스처 (D-010):
  - 동물·식물·이끼 → 클릭
  - 지형(벽지·땅·물) → 클릭·드래그 페인트
  - 식물·이끼는 `plant_surfaces` 면만 허용
- 배치 가능 영역 표시
- 선택 오브젝트의 미리보기 표시
- 변경 전·후 예상 안정도 비교
- 되돌리기와 완료 버튼
- 편집 중 시간 진행 여부 표시

## UI-05 생물 목록

카드 정보:

- 도트 이미지
- 이름
- 식성
- 환경 요구 요약
- 활동 구역
- 사회성
- 현재 사육장 적합도
- 가장 위험한 기존 생물

필터:

- 식성
- 환경 성향
- 활동 구역
- 사회성
- 안전하게 배치 가능

## UI-06 배치 전 진단

표시 항목:

| 항목 | 요구 조건 | 현재 상태 | 판정 |
|---|---|---|---|
| 온도 | 범위 | 현재값 | 적합 여부 |
| 습도 | 범위 | 현재값 | 적합 여부 |
| 공간 | 필요량 | 남은 양 | 적합 여부 |
| 은신처 | 요구량 | 현재 수 | 적합 여부 |
| 합사 | 제한 | 기존 생물 | 안전·주의·위험 |

행동:

- 현재 사육장에 배치
- 환경 먼저 수정
- 다른 사육장 추천
- 위험을 감수하고 배치
- 취소

## UI-08 긴급 오버레이

- 게임 자동 감속 상태
- 사건 이름과 위험 단계
- 영향을 받는 생물
- 문제 위치 강조
- 원인 요약
- 사고 진행 단계
- 생물 분리, 이동, 격리, 먹이 공급, 장비 작동, 청소

## UI-09 배경화면 모드

평상시:

- 메뉴와 수치 비노출
- 자연스러운 도트 애니메이션
- 기본 마우스 입력 통과

주의:

- 작은 상태 아이콘
- 문제 사육장 위치 표시

긴급:

- 자동 감속
- 짧은 경고
- 문제 대상 표시
- 클릭 또는 단축키로 일반 화면 전환

## 공통 컴포넌트 상태

- 버튼: 기본, 호버, 선택, 비활성, 위험, 로딩
- 사육장 카드: 안정, 주의, 위험, 긴급, 선택
- 생물 카드: 배치 가능, 부적합, 합사 주의, 포식 위험
- 경고: 정보, 주의, 위험, 긴급, 해결
- 수치 표시: 정상 범위, 미달, 초과, 생존 범위 이탈

## 접근성 원칙

- 색상만으로 상태를 표현하지 않음
- 상태 아이콘에 텍스트 라벨 제공
- 작은 도트 그래픽과 UI 텍스트의 명도 대비 확보
- 클릭 영역은 도트 이미지 크기보다 크게 설정
- 긴급 애니메이션은 과도한 점멸을 사용하지 않음
